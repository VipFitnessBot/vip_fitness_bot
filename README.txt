# Fitness Club Parkova VIP Bot

Telegram бот для системи підписки та бонусів фітнес-клубу.

## Функції:
- Інформація про абонементи та рівні знижок
- Бонуси для підписників
- Оплата через WayForPay (посилання)
- Кнопкове меню

## Запуск локально:
1. Встановити залежності: `pip install -r requirements.txt`
2. Запустити: `python bot.py`
3. BOT_TOKEN має бути заданий як змінна середовища
